# SDL3-CS.Android.Image

This package contains native sdl_image runtime libraries for sdl3-cs.

## Version Information

| Package | Native library version |
|---------|------------------------|
| SDL3-CS.Android.Image | SDL_image 3.4.4 |

## Platform Support

This package is scoped to Android and contains native artifacts only for these RIDs: android-arm, android-arm64, android-x86, android-x64.

Use it together with SDL3-CS. Add-on packages may also require the base platform package $(@{Id=Android; PackageBase=SDL3-CS.Android; Rids=System.Object[]}.PackageBase).
